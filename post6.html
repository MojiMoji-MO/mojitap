스마트폰에서는 키보드에 있는 웃는 얼굴 아이콘만 누르면 이모지를 쉽게 쓸 수 있습니다. 하지만 블로그 글을 쓰거나, 업무용 이메일을 보낼 때, 혹은 PC 카카오톡을 사용할 때 이모지를 넣고 싶어 당황한 적 없으신가요? 윈도우와 맥(Mac)에는 1초 만에 이모지 창을 띄울 수 있는 숨겨진 전용 단축키가 있습니다. 이 단축키 하나만 외워두면 여러분의 디지털 작업 속도가 비약적으로 상승합니다.

On a smartphone, inserting emojis is as simple as tapping the smiley face icon on the keyboard. But have you ever felt lost when trying to add emojis while writing a blog post, sending a professional email, or chatting on your PC? Both Windows and Mac have hidden shortcut keys that bring up the emoji panel in just 1 second. Memorizing this single shortcut will drastically improve your digital workflow.

1. 윈도우(Windows 10, 11) 이모지 단축키

1. Windows 10 & 11 Emoji Shortcuts

윈도우 사용자라면 아주 간단한 단축키 조합으로 이모지 패널을 호출할 수 있습니다. 텍스트를 입력할 수 있는 곳(메모장, 웹 브라우저 주소창 등)을 클릭한 뒤 아래의 키를 눌러보세요.

If you are a Windows user, you can summon the emoji panel with a very simple key combination. Click anywhere you can type text (like Notepad or a web browser address bar) and press the following keys.

  • 기본 단축키: Win (윈도우 로고 키) + . (마침표)
  • 보조 단축키: Win (윈도우 로고 키) + ; (세미콜론)
  • Primary Shortcut: Win (Windows Logo Key) + . (Period)
  • Alternative Shortcut: Win (Windows Logo Key) + ; (Semicolon)

2. 맥북(macOS) 이모지 단축키

2. Mac (macOS) Emoji Shortcuts

맥(Mac) 환경에서는 윈도우와 키보드 배열이 다르기 때문에 다른 단축키를 사용해야 합니다. 손가락 세 개를 이용하여 아래 단축키를 동시에 누르면 '문자 뷰어(Character Viewer)'가 나타납니다.

Since Mac keyboard layouts differ from Windows, you need to use a different shortcut. Press the following keys simultaneously with three fingers to open the 'Character Viewer'.

  • 맥북 단축키: Control (⌃) + Command (⌘) + Space bar
  • Mac Shortcut: Control (⌃) + Command (⌘) + Space bar

Windows 10 / 11 Windows 10 / 11 Win + . / Win + ; 지원 (한글/영문 검색 가능) Supported (Multi-language)
macOS macOS Ctrl + Cmd + Space 지원 (우측 상단 돋보기 아이콘) Supported (Search icon on top right)

3. 단축키가 갑자기 안 먹힐 때 대처법

3. What to Do When Shortcuts Don't Work

PC에서 작업을 하다가 단축키를 눌러도 이모지 창이 뜨지 않는다면 다음 세 가지를 확인해 보세요.

If the emoji panel doesn't appear when you press the shortcut while working on your PC, check these three things.

  1. 입력창 포커스 확인: 반드시 텍스트를 입력할 수 있는 빈칸(커서가 깜빡이는 곳)을 마우스로 먼저 클릭해야 패널이 열립니다.
  2. 한영 전환 오류: 키보드 입력기가 꼬였을 수 있습니다. 한/영 키 또는 Caps Lock 키를 눌러 입력 상태를 전환한 후 다시 시도해 보세요.
  3. ctfmon.exe 실행 (윈도우 전용): 윈도우 키 + R을 눌러 실행 창을 띄운 뒤, ctfmon.exe를 입력하고 엔터를 치면 키보드 입력기가 초기화되어 해결될 수 있습니다.
  1. Check Input Focus: The panel will only open if you first click inside a text field where the cursor is blinking.
  2. Language Toggle Error: The keyboard input system might be stuck. Try toggling your language key or Caps Lock, then try again.
  3. Run ctfmon.exe (Windows Only): Press Win + R, type ctfmon.exe, and press Enter. This resets the keyboard input process.
